Mist iteration atlas

EchoesOfAethon — layered WebGL mist stack (R3F / drei / three). Companion doc for tuning render order, density, and debug flags.

Render order ladder

Within the transparent queue, lower renderOrder is drawn earlier (painter’s back → front). Opaque terrain and props usually render in the opaque pass, so they are not strictly interleaved with these numbers.

SkyFogDome — shader sphere, distant color wash−100
CelestialBodies — moon / stars meshes (fog=false)−90 … −87
SkyCloudLayer — billboard planes (forward of dome / celestials)−50
GroundFogLayer — stacked horizontal noise sheets−3
RollingFogBands — low drifting banks−2
VolumetricFogCurtains — vertical haze planes (same order tier as bands)−2
MistParticles — point sprites (wind in VS)0 (default)
CloudFogVolume — drei Clouds + Cloud, MeshBasicMaterial5
World & props — GeneratedMap, interactables (opaque / default)—

Layer cards

Sky

SkyFogDome

Inverted or oversized sphere with custom shader; establishes dusk tint and horizon falloff. Kept at renderOrder: -100 so it paints first, behind the stack.

Sky

CelestialBodies

Optional. Materials use fog=false so scene fog does not grey out moon/stars; sky clouds still sit forward on the ladder to veil them artistically.

Sky

SkyCloudLayer

Billboard planes, high relative order vs dome/celestials. Reads as soft cloud cards above the playfield.

Floor

GroundFogLayer

Horizontal planes with FBM noise; player and lantern radial clears (smooth pockets), optional edge fade toward map bounds. Multi-layer stack for depth.

Volume

CloudFogVolume

@react-three/drei Clouds / Cloud with MeshBasicMaterial; stylized volumetric puff layers at configured heights and opacities.

Floor

RollingFogBands

Many low planes, drift animation, noise variation; stronger alpha than curtains. Lantern scatter tint can break uniformity.

Depth

VolumetricFogCurtains

Tall thin planes, very low alpha; suggest depth without readable rectangles. Gentle drift and noise.

Grain

MistParticles

THREE.Points + custom shaders: wind in vertex stage, height fade, circular falloff in fragment shader; shares clear-pocket idea with ground fog.

FBM loop (fragment-friendly)

float fbm(vec2 p) {
  float v = 0.0;
  float a = 0.5;
  mat2 m = mat2(1.6, 1.2, -1.2, 1.6);
  for (int i = 0; i < 5; i++) {
    v += a * noise(p);
    p = m * p;
    a *= 0.5;
  }
  return v;
}

Smoothstep pocket (player / lantern clear)

float d = distance(worldXZ, centerXZ);
float pocket = smoothstep(inner, outer, d);
// pocket ≈ 0 near center (clear), 1 in fog
float alpha = baseAlpha * pocket;

Height falloff (vertical mist)

float h = 1.0 - smoothstep(yLow, yHigh, worldY);
// or on local pos.y for particles:
float vHeightFade = 1.0 - smoothstep(0.12, 2.35, pos.y);

Each octave shrinks amplitude (a *= 0.5) and rotates/scales UVs so repeats are less obvious. World-space p should be offset by time for drift. Ground fog in this project uses fixed world-scale multipliers on top of config noiseScale.

Combine FBM with pockets: mix(thin, thick, fbm(p)) * pocket for readable silhouettes near the player.

Scene assembly note (PrototypeScene)

Fog-related nodes are ordered after atmosphere and sky stack, then map geometry: GroundFogLayer → optional CloudFogVolume → conditional RollingFogBands / VolumetricFogCurtains → MistParticles when showParticles is on.

Design intent (quick read)

Dome and curtains sell depth; ground sheets and rolling bands sell local volume; particles add fine grain; drei clouds add soft billows at mid heights. Tune each band independently so total alpha does not mud out silhouettes.
